# セールスファネル・マーケティングオートメーション 調査・分析レポート

## 1. 調査対象サービス分析

### 1.1 ClickFunnels（クリックファネル）

**概要**
- 2014年にラッセル・ブランソンが開発
- 世界No.1ファネルビルダー
- わずか約5年で1億ドルを突破

**価格**
| プラン | 月額 |
|--------|------|
| Basic | $99 |
| Pro (全機能) | $297 |

**主要機能**
- ドラッグ&ドロップビルダー
- 豊富なテンプレート
- ワンクリックアップセル
- ステップメール機能（Actionetics）
- A/Bテスト

**課題**
- 英語ベース（2025年に日本語版リリース予定）
- 価格が高い（円安の影響）
- 日本語サポートが限定的

---

### 1.2 UTAGE（ウタゲ）- 国産オールインワン

**概要**
- 2021年リリースの純国産ツール
- 集客から販売、カスタマーサポートまで対応

**価格**
- 初期費用: 無料
- 月額: 21,670円（税込）
- 14日間無料トライアル

**主要機能（11種類）**
1. ファネル作成機能
2. メール配信機能
3. LINE連携機能
4. 会員サイト作成機能
5. サブスク商品作成機能
6. 自動ウェビナー機能
7. イベント機能
8. コンテンツ販売機能
9. オーダーバンプ機能
10. ワンクリックアップセル機能
11. AIアシスト機能（ChatGPT連携）

**強み**
- 完全日本語対応
- 日本の決済サービスとの連携
- LINE連携が充実
- 日本語サポート

## 5. 競合との差別化ポイント

### 5.1 ClickFunnelsの弱点を突く

| 弱点 | 当システムの対策 |
|------|------------------|
| 高価格（$99-297/月） | 段階的価格設定、無料プラン提供 |
| 日本語対応が不十分 | 完全日本語UI/サポート |
| 日本決済との連携弱い | PAY.JP, GMO-PG, LINE Pay完全対応 |
| LINE連携なし | LINE連携を標準機能として搭載 |

### 5.2 UTAGEとの差別化

| UTAGE | 当システム |
|-------|-----------|
| クローズドソース | オープンアーキテクチャ |
| 固定価格 | 使用量ベース課金も選択可 |
| プラットフォーム依存 | セルフホスト対応 |

### 5.3 独自の強み

1. **オープン設計**: API-first、拡張性重視
2. **AI-Native**: AI機能を全機能に統合
3. **日本市場特化**: LINE、日本決済、日本語完全対応
4. **モダン技術スタック**: 高速、スケーラブル
5. **開発者フレンドリー**: カスタマイズ容易
